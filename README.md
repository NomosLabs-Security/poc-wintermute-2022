- **Full Analysis:** [NomosLabs Security Intelligence Archive](https://nomoslabs.io/archive/wintermute-2022)

## Description

Wintermute's $160M hot wallet used a vanity address generated by the Profanity
tool, which had a cryptographic flaw: it generated private keys from a 32-bit
seed instead of 256-bit random, reducing the keyspace from 2^256 to 2^32.
This made GPU brute-force recovery feasible in days for ~$5,000 compute cost.
